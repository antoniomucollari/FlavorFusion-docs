# Abstract

The rapid expansion of the food delivery industry demands highly scalable, real-time logistics and management platforms. While existing market leaders like Wolt and Lieferando provide robust consumer experiences, they often lack critical transparency tools—such as precise, cost-saving delivery distance metrics. Furthermore, building systems that can handle sudden traffic spikes requires architectures specifically designed for horizontal scalability. This thesis presents the development of *Flavor Fusion*, a comprehensive, production-ready food delivery platform engineered to address these operational and architectural limitations.

The proposed system is built on a stateless backend architecture to ensure seamless horizontal scaling under high loads. The technical foundation integrates PostGIS for advanced spatial data management, Redis for high-performance caching, and RabbitMQ for reliable asynchronous message brokering. To validate the platform with realistic scale and complexity, the database was populated with real-world restaurant and menu data web-scraped from Wolt. Additionally, a versatile payment integration system was implemented to support frictionless user interactions, including secure "save card" functionality for future checkouts.

A key innovation of this work is the integration of an open-source routing matrix to enhance transparency in delivery logistics. This feature uniquely empowers customers by calculating and displaying the exact driver delivery distance in kilometers when they are browsing different restaurant branches. By exposing these precise routing details, the platform enables informed, cost-saving purchasing decisions for the end-user—a level of transparency that is currently absent in major incumbent platforms. To optimize performance and minimize routing calculation overhead, the system leverages Redis to cache geospatial POINTS and distance matrix results.

The system's performance, resilience, and horizontal scalability were rigorously evaluated through comprehensive load testing. Under high-stress scenarios simulating significant concurrent user traffic, the stateless architecture demonstrated exceptional reliability, achieving consistently fast, sub-second response times without any failed requests. Ultimately, this research demonstrates that combining advanced spatial routing, intelligent caching mechanisms, modern message brokering, and a stateless architecture yields a highly performant, cost-effective solution that enhances transparency for consumers while maintaining a seamless user experience.
